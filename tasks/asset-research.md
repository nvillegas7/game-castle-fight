### WAV to OGG Conversion
Many packs provide WAV. Convert before importing:
```bash
# Install ffmpeg if needed: brew install ffmpeg
# Convert all WAV to OGG in a directory:
for f in *.wav; do ffmpeg -i "$f" -c:a libvorbis -q:a 4 "${f%.wav}.ogg"; done
```

### Font Integration (Godot 4)
1. Place .ttf/.otf in `res://assets/fonts/`
2. Create `FontFile` resource
3. Set as theme override in UI nodes
4. For pixel fonts: disable filtering (Import > Filter = false)
